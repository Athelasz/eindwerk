convert_excel
*************

convert_excel.inv_select(df)

   inv_select Per meeting collect all attendees in one  list of
   dictionaries

   Parameters:
      **df** (*DataFrame*) – dataframe to fetch data from

   Returns:
      nested dict with all attendees per meeting name

   Return type:
      dict

convert_excel.main(tzone_str='Europe/Brussels', sheet='list_webex.xlsx')

   main Converts an excel file to a dict suitable for creating webex
   meetings via the Webex REST api

   Parameters:
      * **tzone_str** (*str**, **optional*) – desired timezone,
        defaults to ‘Europe/Brussels’

      * **sheet** (*str**, **optional*) – desired source file,
        defaults to ‘list_webex.xlsx’

   Returns:
      a nested dict of unique meetings + participants

   Return type:
      dict

convert_excel.parse_time(cols, df, tzo)

   parse_time Function to parse strings to time zone aware  datetime
   objects, and then convert them to ISO8601 format strings

   Parameters:
      * **cols** (*list*) – list of time columns

      * **df** (*DataFrame*) – source df to process

      * **tzo** (*string*) – time zone

   Returns:
      list of modified columns

   Return type:
      list
